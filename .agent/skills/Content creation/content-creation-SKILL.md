# Content Creation Skill — Social Media (Instagram & Facebook)

This skill guides the creation of social media content that is consistent with the project's visual identity and brand voice. Before writing any content, always read the available brand assets in the project. The content should feel like a natural extension of the website and graphic profile — not a separate entity.

---

## 1. Before You Write Anything — Read the Brand Assets

Always locate and review these files in the project before generating content:

| Asset | What to look for |
| :--- | :--- |
| **Graphic profile / Brand guidelines** | Tone of voice, brand values, dos and don'ts, typography rules |
| **Color palette** | Primary, secondary, and accent colors — use hex codes in descriptions |
| **Logotype** | How the logo is used, clear space rules, what NOT to do with it |
| **Web design / Mockups** | Headline style, button copy style, overall tone of existing content |

If these files exist in the project, read them first. The content you generate must match what already exists in the brand's visual and written language.

---

## 2. Brand Voice & Tone

Extract the brand voice from the guidelines. If no explicit tone is documented, infer it from the website copy and mockups. Use this framework:

**Define the tone on these axes:**
- Formal ←→ Casual
- Serious ←→ Playful
- Technical ←→ Accessible
- Reserved ←→ Bold

**Document the result like this (fill in per project):**

```
Brand Voice: [e.g. "Warm, professional, and straightforward. Never corporate or stiff."]
Avoid: [e.g. "Slang, excessive exclamation marks, passive voice"]
Always: [e.g. "Use 'du/ni' form in Swedish. Use active verbs. Keep sentences short."]
```

---

## 3. Language Rules

All content must be created in both **Swedish** and **English** unless specified otherwise. Follow these rules:

### Swedish
- Use **du-tilltal** (informal you) unless the brand is explicitly formal
- Short sentences — max 15–20 words per sentence in captions
- Avoid direct translations from English — write natively in Swedish
- Use Swedish punctuation correctly (no Oxford comma)
- Hashtags: mix Swedish and English hashtags (Swedish for local reach, English for broader reach)

### English
- Use **international English** (avoid heavy Americanisms or British idioms unless brand-specific)
- Mirror the same tone as the Swedish version — do not make the English version more formal or more casual
- Hashtags: English only

### Output format for bilingual content:
Always deliver both versions together, clearly labelled:

```
🇸🇪 SVENSKA
[Swedish caption here]

🇬🇧 ENGLISH
[English caption here]
```

## 4. Instagram Content

### 4.1 Caption Structure
Every Instagram caption should follow this structure:

1. **Hook** (line 1) — the first line must stop the scroll. Ask a question, make a bold statement, or lead with the most interesting thing. This is what shows before "...more".
2. **Body** — 2–4 sentences expanding on the hook. Tell a story, share value, or explain the post.
3. **CTA (Call to Action)** — one clear action. Link in bio, save this post, comment below, tag a friend.
4. **Hashtags** — placed after a line break or hidden with dots (see below).

```
// Instagram caption template

[Hook — one punchy sentence that works without context]

[Body — 2-4 sentences. Keep it conversational.]

[CTA — one action]

.
.
.
#hashtag1 #hashtag2 #hashtag3
```

### 4.2 Instagram Hashtag Strategy

Use **15–25 hashtags** per post. Mix these tiers:

| Tier | Size | Example | Purpose |
| :--- | :--- | :--- | :--- |
| **Niche** | < 50K posts | `#stockholmföretag` | High chance of being found |
| **Mid** | 50K–500K posts | `#webdesign` | Balanced reach |
| **Broad** | 500K–2M posts | `#design` | Visibility boost |

Avoid hashtags with 2M+ posts — your content gets buried instantly.

### 4.3 Instagram Post Types & Tone

| Format | Tone | Caption Length |
| :--- | :--- | :--- |
| **Single image** | Direct, confident | Short — 3–6 lines |
| **Carousel** | Educational, step-by-step | Medium — introduce the carousel, tease the slides |
| **Reel** | Energetic, hook-first | Short — max 5 lines, strong CTA |
| **Story (text)** | Casual, conversational | Very short — 1–2 sentences max |

### 4.4 Instagram Visual Direction (for image briefs)

When briefing visuals or describing image requirements, reference the brand's color palette and graphic profile. Use this format:

```
Visual brief:
- Primary color: [hex from palette]
- Background: [color or style from guidelines]
- Typography: [font name from brand guidelines]
- Logo placement: [e.g. bottom right, within safe zone]
- Mood: [e.g. clean, minimal, warm, bold]
- Avoid: [e.g. busy backgrounds, drop shadows, comic sans]
```

## 5. Facebook Content

### 5.1 Caption Structure

Facebook allows and rewards longer, more conversational content than Instagram. Structure posts like this:

1. **Opening line** — a question or statement that creates curiosity or relevance
2. **Body** — 2–5 sentences. Facebook users read more, so add context and personality
3. **CTA** — direct and specific ("Comment below", "Click the link", "Share with someone who needs this")
4. **Link** (if applicable) — paste the URL on its own line; Facebook auto-generates a preview card
5. **Hashtags** — use only 3–5 on Facebook. More looks spammy.

```
// Facebook post template

[Opening question or statement]

[Body — 2-5 sentences with more context than Instagram]

[CTA]

[URL if applicable]

#hashtag1 #hashtag2 #hashtag3
```

### 5.2 Facebook Post Types

| Format | Best for | Notes |
| :--- | :--- | :--- |
| **Link post** | Driving traffic to website | Let Facebook generate the preview card — don't add the URL in the text |
| **Image post** | Brand awareness, announcements | Same visual rules as Instagram |
| **Video** | Engagement, storytelling | Add captions — 85% of Facebook video is watched without sound |
| **Event** | Launches, webinars, IRL events | Fill all fields — description, location, cover image |

### 5.3 Facebook vs Instagram — Key Differences

| | Instagram | Facebook |
| :--- | :--- | :--- |
| Caption length | Short–medium | Medium–long |
| Hashtags | 15–25 | 3–5 |
| Tone | Visual-first, punchy | Conversational, contextual |
| Links in caption | Not clickable — use "link in bio" | Clickable — paste directly |
| Best content time | Mornings & evenings | Lunch & early afternoon |

## 8. AI Image Generation — Prompts Based on Brand Assets

Since the image generation tool may vary per project, this section covers how to build on-brand prompts for the three most common AI tools: **Midjourney**, **DALL-E (ChatGPT)**, and **Adobe Firefly**. The prompt structure is the same — only the syntax differs slightly per tool.

### 8.1 Before Writing Any Image Prompt

Always extract these four things from the brand assets before generating a prompt:

```
1. COLOR PALETTE     → List the 2-3 primary hex colors (e.g. #1A1A2E, #E8D5B7)
2. VISUAL STYLE      → What does the brand look like? (e.g. minimalist, bold, warm photography, flat illustration)
3. MOOD / FEELING    → What emotion should the image convey? (e.g. trustworthy, energetic, calm, premium)
4. WHAT TO AVOID     → From the graphic profile dos and don'ts (e.g. no busy backgrounds, no stock photo clichés)
```

### 8.2 Universal Prompt Structure

Use this structure regardless of which tool you use. Fill in each block from the brand assets.

```
[SUBJECT] — what is in the image (person, product, scene, abstract)
[STYLE] — visual style from brand guidelines
[MOOD] — emotional tone
[COLOR] — dominant colors from palette
[LIGHTING] — lighting style (soft natural light, studio, golden hour, etc.)
[COMPOSITION] — framing (centered, rule of thirds, close-up, wide shot)
[NEGATIVE] — what to exclude (busy backgrounds, text, logos, watermarks)
[FORMAT] — aspect ratio for the platform
```

**Example prompt (filled in):**
```
A professional woman working at a modern desk, minimalist Scandinavian interior design,
calm and trustworthy mood, warm beige and deep navy tones (#E8D5B7 and #1A1A2E),
soft natural window light from the left, rule of thirds composition with space on the right,
no text, no logos, no cluttered backgrounds, 4:5 aspect ratio
```

### 8.3 Platform Format & Aspect Ratios

Always specify the correct ratio at the end of every prompt.

| Platform & Format | Ratio | Pixels |
| :--- | :--- | :--- |
| Instagram Feed (portrait) | 4:5 | 1080 × 1350px |
| Instagram Feed (square) | 1:1 | 1080 × 1080px |
| Instagram Feed (landscape) | 1.91:1 | 1080 × 566px |
| Instagram Story / Reel cover | 9:16 | 1080 × 1920px |
| Facebook Feed image | 1.91:1 | 1200 × 630px |
| Facebook Story | 9:16 | 1080 × 1920px |

### 8.4 Tool-Specific Syntax

#### Midjourney
Add parameters at the end of the prompt after `--`.

```
[Your prompt here] --ar 4:5 --style raw --v 6
```

Useful Midjourney parameters:
- `--ar 4:5` — aspect ratio
- `--style raw` — less stylized, more realistic
- `--no text, logos` — exclusions (replaces negative prompts)
- `--v 6` — latest model version

**Full Midjourney example:**
```
Minimalist flat lay of a coffee cup and notebook on a warm beige linen surface,
Scandinavian design aesthetic, calm and focused mood, warm beige and muted terracotta tones,
soft diffused overhead light, centered composition, clean background --ar 1:1 --style raw --v 6 --no text logos watermarks clutter
```

---

#### DALL-E (via ChatGPT)
Write as a clear, descriptive sentence. Be specific — DALL-E responds well to detailed, natural language.

```
Create a [ratio] image of [subject]. The style should be [visual style].
The mood is [mood]. Use a color palette of [colors from brand].
Lighting: [lighting]. Composition: [composition].
Do not include any text, logos, or [other exclusions].
```

**Full DALL-E example:**
```
Create a 4:5 portrait image of a minimalist home office setup with a wooden desk,
a plant, and a laptop. The style should be clean Scandinavian interior photography.
The mood is calm, productive, and premium. Use warm beige, off-white, and deep navy tones.
Soft natural light coming from a window on the left. Slightly off-center composition.
Do not include any text, people, logos, or busy backgrounds.
```

---

#### Adobe Firefly
Works similarly to DALL-E. Use the **Style** and **Color tone** fields in the UI in addition to your text prompt — these map directly to brand assets.

```
[Subject and scene description], [visual style], [mood], [color description], [lighting], [composition]
```

In Firefly's UI additionally set:
- **Style**: match to brand (Photography / Graphic / Art)
- **Color tone**: pick the closest match to brand palette
- **Lighting**: match brand mood
- **Aspect ratio**: select from the format table above

**Full Firefly example:**
```
A close-up of hands holding a warm coffee cup, cozy and minimal setting,
lifestyle photography style, calm and inviting mood, warm beige and soft brown tones,
soft natural light, shallow depth of field, centered close-up composition
```

### 8.5 Prompt Templates per Post Type

Ready-to-use starting points. Replace `[BRAND STYLE]`, `[BRAND COLORS]`, and `[BRAND MOOD]` with values from the project's graphic profile.

**Product / Service highlight:**
```
A clean, minimal [product/service scene], [BRAND STYLE] aesthetic,
[BRAND MOOD] mood, [BRAND COLORS] color palette,
soft studio lighting, centered product composition with negative space,
no text, no people, no logos, 1:1 aspect ratio
```

**Lifestyle / People:**
```
A [age/gender] person [action relevant to brand], [BRAND STYLE] environment,
[BRAND MOOD] atmosphere, [BRAND COLORS] tones throughout,
natural soft lighting, candid lifestyle photography feel,
no logos, no text overlays, 4:5 aspect ratio
```

**Abstract / Brand texture:**
```
Abstract background texture with [BRAND COLORS] color palette,
[BRAND STYLE] visual language, [BRAND MOOD] feeling,
smooth gradients and subtle geometric shapes, minimal and clean,
suitable as a social media background, no text, 1:1 aspect ratio
```

**Quote / Text background:**
```
A simple, minimal background for a text overlay post, [BRAND COLORS] palette,
soft gradient or subtle texture, [BRAND STYLE] design language,
low visual complexity so text remains legible, no existing text or logos,
1:1 or 4:5 aspect ratio
```

### 8.6 What to Avoid in AI-Generated Images

Always include these exclusions in prompts to prevent common AI mistakes:

```
❌ No text or typography (AI-generated text is almost always broken)
❌ No watermarks
❌ No extra limbs or distorted hands (use close-up crops to avoid hands)
❌ No logos (add the real logo in Canva/Figma after generation)
❌ No stock-photo-style forced smiles
❌ No busy or cluttered backgrounds (unless the brand calls for it)
```

## 9. Quality Checklist

Before delivering any content, verify:

**Brand alignment**
- [ ] Tone matches the brand voice extracted from guidelines
- [ ] Colors and visual references match the palette
- [ ] Logo usage follows the graphic profile rules

**AI Image prompts**
- [ ] Prompt includes correct aspect ratio for the platform (see Section 8.3)
- [ ] Brand colors referenced from palette (use hex codes)
- [ ] Visual style matches graphic profile
- [ ] Negative exclusions included (no text, no logos, no distorted hands)
- [ ] Generated image reviewed — add real logo in Canva/Figma after generation

**Copy quality**
- [ ] Hook works as a standalone sentence (no context needed)
- [ ] No passive voice — rewrite to active
- [ ] One CTA only — not multiple competing actions
- [ ] No spelling errors in either language
- [ ] Swedish is written natively, not translated from English

**Platform rules**
- [ ] Instagram: 15–25 hashtags, short caption, hashtags separated from body text
- [ ] Facebook: 3–5 hashtags, longer body, link pasted on its own line
- [ ] Both versions delivered and clearly labelled (🇸🇪 / 🇬🇧)

**Bilingual consistency**
- [ ] Both versions communicate the same message
- [ ] Neither version is noticeably more formal or casual than the other
- [ ] Hashtag strategy differs per platform and language (see Section 4.2)
